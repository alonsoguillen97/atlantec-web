# ATLANTEC Favicon Files

Este directorio contiene los archivos de favicon para ATLANTEC:

- `favicon.svg` - Favicon principal en formato SVG (recomendado para navegadores modernos)
- `favicon.ico` - Favicon para compatibilidad con navegadores antiguos  
- `apple-touch-icon.png` - Icono para dispositivos Apple (180x180)
- `favicon-32x32.png` - Favicon PNG 32x32 para alta resolución
- `favicon-16x16.png` - Favicon PNG 16x16 estándar

## Colores Corporativos ATLANTEC
- Azul Atlántico: #1e40af
- Turquesa Marino: #36d1dc  
- Naranja Sol: #ff751f

## Elementos del diseño
- Letra "A" estilizada representando ATLANTEC
- Ondas representando el océano Atlántico
- Punto tecnológico simbolizando la innovación
- Gradiente corporativo de fondo
